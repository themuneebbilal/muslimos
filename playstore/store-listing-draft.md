# Store Listing Draft

## App Name

MuslimOS

## Short Description

Prayer times, Quran, hadith, worship tools, qibla, and journal in one calm Islamic app.

## Full Description

MuslimOS brings your daily Islamic essentials into one focused experience.

Use MuslimOS to:

- check prayer times
- read the full Quran
- listen to recitation with synced highlighting
- save favorite surahs and bookmarks
- explore hadith collections
- use worship and dhikr tools
- find qibla direction
- keep a private journal
- track Islamic calendar moments

Built with a calm emerald and gold design language, MuslimOS is designed to feel intentional, readable, and respectful across both phone and larger screens.
