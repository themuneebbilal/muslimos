# Store Assets

Prepare these before Play Console submission.

## Required

- app icon
  512 x 512 PNG
- feature graphic
  1024 x 500 PNG
- phone screenshots
  At least 2, ideally 6 to 8
- privacy policy URL

## Recommended screenshot set

- Home
- Prayer Times
- Quran list
- Quran reading with audio highlight
- Worship
- Hadith
- Qibla
- Journal

## Brand direction

- preserve MuslimOS emerald/gold/cream palette
- use Amiri + DM Sans
- avoid generic Android marketing mockups that do not match the actual UI

## File naming convention

- `phone-home.png`
- `phone-prayer-times.png`
- `phone-quran-list.png`
- `phone-quran-reader.png`
- `phone-worship.png`
- `phone-hadith.png`
- `phone-qibla.png`
- `phone-journal.png`
- `feature-graphic.png`
- `app-icon-512.png`
